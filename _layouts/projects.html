Deja Photo

  • Android application developed in a team of 6 for Software Engineering at UCSD
  • Changes phone wallpaper according to current location, "karma", and more
  • Uses a friends list for users to share photos with each other and give karma
  • Photos stored in a Firebase database, with an account system for individual data

Platforming AI

  • Created a simple platforming game in Python that is played by an AI bot
  • Bot learns through neuroevolution
  • Based on Evolving Neural Networks through Augmenting Topologies by Kenneth O. Stanley and Risto Miikkulainen
  • Work in progress

CV Maze Game

  • Simple maze game where the player controls their character with their hand
  • Computer vision is used to track the hand's position through a webcam
  • Written in Java and Python

Chess

  • Fully playable chess game written in Java
  • Also has a simple AI for single player play
  • Developed as a fun project in high school while learning how to code
